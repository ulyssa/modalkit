## About

This is a Rust library for building TUI applications using the [ratatui] and
[modalkit] crates.

## Usage

This crate can be used by adding `modalkit-ratatui` to your dependencies in
your project's `Cargo.toml`.

```toml
[dependencies]
modalkit-ratatui = "0.0.24"
```
